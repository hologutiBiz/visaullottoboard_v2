Information We Collect

Account & Personal Information

When you create an account or use our Service, we may collect the following:

Account Info

Email address, username, and display name

Authentication Data

Encrypted password, OAuth tokens (Google or Facebook sign-in)

Subscription Info

Payment status, plan type, and subscription expiration date

Profile Preferences

Optional user preferences and notification settings

Automatically Collected Information

  • Device Information: Device type, operating system, and unique device identifiers
  • Usage Data: Features accessed, time spent on the app, and interaction patterns
  • Log Data: IP address, browser type, access times, and pages viewed
  • Firebase Analytics: App performance metrics, crash reports, and user engagement statistics

Feature Usage Data

When you interact with features of the Lotto Forecast App, we may record activity related to the following:

  • Predictions: Forecast content uploaded by our team that you access or save
  • Day Result: Your interactions with real-time Baba Ijebu draw results
  • Game Results: Baba Ijebu game result pages you access (historical data from October 2023 to date)
  • Counting Weeks: Selections and week-tracking configurations you set
  • Moving Number: Number movement preferences and views
  • Track Pattern: Pattern tracking history and preferences you configure
  • Frequent Numbers: Frequency analysis views and saved preferences
  • Lotto Classification Chart: Chart views and interactions
  • Forecast Central Panel: Usage activity on the large-screen integrated dashboard
  • Notification Settings: Your push notification and alert preferences

Data Storage & Security

Where Your Data Is Stored

All personal user data is stored exclusively within Google Cloud infrastructure. We do not store personal user data on any third-party server outside of Google Cloud.

  • Firebase Authentication (Google Cloud): Stores your account credentials and authentication tokens securely
  • Cloud Firestore (Google Cloud Platform): Stores your application data, subscription status, saved preferences, and feature usage records
  • Encryption: All data is encrypted in transit using HTTPS/TLS and encrypted at rest within Google Cloud

Security Measures

SSL/TLS Encryption Bcrypt Password Hashing Firebase App Check Regular Security Audits Role-Based Access Controls

Data Retention

Account Status Retention Period
Active accounts Retained for the duration the account remains active
Inactive accounts Retained for 90 days after last login, then deleted
Deleted accounts Personal data permanently deleted within 30 days of request
Anonymized analytics May be retained for statistical and improvement purposes

App-Specific Information

Android Application

The Lotto Forecast App is available on Android via the Google Play Store. The following device permissions may be requested:

  • Internet Access: Required to fetch live draw results, predictions, and all app data
  • Notifications: Required to deliver draw result alerts, prediction updates, and subscription reminders
  • Storage: Used for local data caching to improve app speed and offline resilience

Google Play Services may independently collect device identifiers and performance metrics in accordance with Google's Privacy Policy.

Web Application

The Lotto Forecast App is also accessible via any modern web browser at app.visuallottoboard.com. Standard browser cookies and session tokens are used to maintain your authenticated state and user preferences across sessions.

Forecast Central Panel

The Forecast Central Panel is a specialized, integrated dashboard available exclusively to subscribers on large-screen devices (desktop and wide-format tablets). It provides a unified view of the Lotto Classification Chart, Baba Ijebu Game Results, and the current Day Result side by side. This interface does not collect any additional personal data beyond standard session analytics.

Sign-In with Facebook

If you choose to sign in using Facebook, we receive your public profile information (name and profile picture), email address, and Facebook user ID for the sole purpose of creating and authenticating your account. We do not post to your Facebook timeline, access your friends list, or use this information for advertising or any purpose beyond account identification.
